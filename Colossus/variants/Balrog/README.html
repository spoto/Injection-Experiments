README for Colossus Balrog Variant

What is Balrog ?

Balrog is the same as Default with the only addition of the Balrog. A Balrog is a non-summonable 12-3 magic rangestriker. One is put in a player home Tower for every 300 pts awarded to the player, and can be recruited by any one mustering in that Tower.

Implementation in Colossus is a bit ugly, with 6 different Balrog visible in the caretaker. Sorry about that, this may (or may not) change in the future.
