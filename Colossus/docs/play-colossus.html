Play Colossus (Titan) Online!

If you were looking for a way how to play Titan online, then you are (almost) right here!

"Colossus is a Java clone of the boardgame Titan, a turn-based fantasy wargame for 2-6 players. Each player moves stacks of creatures around a strategic board, recruiting more creatures and fighting enemy stacks on tactical maps."

So, with Colossus you can basically play Titan with your Computer. The Public Game Server serves two purposes:

  1. Find opponents to play against
  2. Games are hosted on the server, so no player needs to fiddle with his firewall or NAT settings
Both of above have been traditionally the main obstacles how to be able to play Colossus online.

However, this here (play-colossus.net) is "only" the server that will run the actual games; but all documents and the different possibilities how to start the game are hosted/provided on our SourceForge project page.

This page here is just for information purposes, if somebody hears about the "play-colossus.net" server and e.g. tries to connect to here directly with his web browser: Nope - nothing to do or see here ;-)
